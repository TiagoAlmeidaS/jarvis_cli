# @openai/jarvis-responses-api-proxy

<p align="center"><code>npm i -g @openai/jarvis-responses-api-proxy</code> to install <code>jarvis-responses-api-proxy</code></p>

This package distributes the prebuilt [jarvis Responses API proxy binary](https://github.com/openai/jarvis/tree/main/jarvis-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/jarvis-responses-api-proxy.js --help
```

Refer to [`jarvis-rs/responses-api-proxy/README.md`](https://github.com/openai/jarvis/blob/main/jarvis-rs/responses-api-proxy/README.md) for detailed documentation.
